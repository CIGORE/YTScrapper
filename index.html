🚀 Cara Mendapatkan API Key (Gratis)

1
Buka Google Cloud Console → buat project baru (atau pilih yang ada)
2
Cari "YouTube Data API v3" di Library → klik Enable
3
Pergi ke APIs & Services → Credentials → klik Create Credentials → API Key → copy key-nya
4
Paste API Key di bawah, masukkan URL channel, lalu klik Scrape Channel
